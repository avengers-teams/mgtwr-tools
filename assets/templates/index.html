数据分析模块使用说明

当前分析页已按 mgtwr-master 实际接口对齐，支持 GWR、MGWR、GTWR、MGTWR 四类模型，并开放线程数、截距项、坐标转换、搜索输出、耗时输出等设置。

GWR

只考虑空间异质性，适合横截面空间数据。

MGWR

不同变量拥有不同空间带宽，适合空间尺度差异明显的情形。

GTWR

同时考虑空间与时间权重，适合面板或时空样本。

MGTWR

不同变量拥有不同空间带宽和时空尺度，表达能力最强，耗时也最高。

页面流程

  • 先导入输入 Excel，再选择输出文件。
  • 选择因变量、自变量和 2 个坐标列。
  • 如果模型是 GTWR 或 MGTWR，还必须选择时间列。
  • 选择模型、核函数、固定带宽与评价准则后，再填写该模型对应参数。
  • 分析完成后，输出文件会包含 summary、coefficients、settings 等工作表。

通用参数

参数 作用 建议
kernel 核函数，可选 gaussian、bisquare、exponential。 默认优先尝试 bisquare 或 gaussian。
fixed 是否固定带宽。True 为距离带宽，False 为自适应近邻带宽。 点位分布不均时，通常优先用 False。
criterion 搜索准则，可选 AICc、AIC、BIC、CV。 小样本优先 AICc。
thread 并行线程数，对应模型中的 thread 参数。 建议从 1、2、4 逐步测试，线程过多不一定更快。
constant 是否自动加入截距项。 通常保持开启，除非输入数据已手动包含常数列。
convert 是否把经纬度投影为平面坐标。 坐标列是真实经纬度时建议开启。
verbose 是否在控制台输出搜索过程。 调参阶段建议开启。
time_cost 是否输出搜索耗时。 建议开启，便于观察线程和参数对速度的影响。

GWR 参数

参数 说明
bw_min / bw_max带宽搜索范围。
tol搜索收敛公差。
bw_decimal带宽保留的小数位数。
max_iter黄金分割搜索最大迭代次数。

MGWR 参数

参数 说明
bw_min / bw_max全局初始化搜索范围。
init_bw初始化带宽；为空时自动从 GWR 派生。
multi_bw_min / multi_bw_max各变量的多带宽范围。可以填单个值，也可以填逗号分隔的多个值。
tol带宽搜索收敛公差。
tol_multi多带宽回代收敛公差。
bws_same_times带宽连续多少轮不变后停止回代。
rss_score是否用 RSS 作为回代评分标准。
n_chunks推断阶段分块数，越大越省内存。
skip_calculate跳过 CCT/ENP 等推断统计，速度更快但输出统计量更少。

GTWR 参数

参数 说明
bw_min / bw_max空间带宽搜索范围。
tau_min / tau_max时空尺度搜索范围。
tol联合搜索收敛公差。
bw_decimal带宽精度。
tau_decimal时空尺度精度。
max_iter联合搜索最大迭代次数。

MGTWR 参数

参数 说明
bw_min / bw_max单一空间带宽搜索范围。
tau_min / tau_max单一时空尺度搜索范围。
init_bw / init_tau初始化带宽与时空尺度；为空时自动从 GTWR 派生。
multi_bw_min / multi_bw_max各变量空间带宽范围。
multi_tau_min / multi_tau_max各变量时空尺度范围。
tol单一带宽与时空尺度搜索收敛公差。
tol_multi多尺度回代收敛公差。
bw_decimal / tau_decimal带宽与时空尺度的小数位精度。
rss_score是否使用 RSS 评价每轮回代。
n_chunks推断阶段分块数。
skip_calculate跳过推断统计项。

输出结果说明

  • summary：模型摘要、主要统计量、搜索结果。
  • coefficients：局部回归系数、预测值、残差，以及可用时的标准误和 t 值。
  • settings：本次运行的全部参数记录。
  • search_scores / bw_history / tau_history：多尺度模型的搜索历史。
